# AI Companion: Context Documentation Router

## Audience
- AI assistants generating cross-feature context.
- Engineers looking for shared vocabulary and taxonomy anchors.

## Purpose
- Entry point for cross-domain context and taxonomy references.

## Belongs Here
- Taxonomy/glossary references used across features.
- Orientation context packs for humans and AI assistants.
- Shared concept maps that reduce cross-doc ambiguity.

## Does Not Belong Here
- Operational runbooks.
- Contributor coding workflows.
- ADR decision records.
