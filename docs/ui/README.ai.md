# AI Companion: UI Documentation Router

## Audience
- AI assistants answering UI behavior and UX implementation questions.

## Purpose
- Entry point for UI contracts and renderer-focused documentation.

## Belongs Here
- UI behavior contracts and UX conventions.
- Frontend implementation notes that affect renderer behavior.
- UI-specific guidance for feature delivery and review.

## Does Not Belong Here
- Broad architecture baselines outside UI scope.
- Runtime operations procedures.
- Contributor workflow docs that are not UI-specific.
